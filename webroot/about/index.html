Original - Date: Dec 16 2018

When I first got a port forwarded to my Dell R710, I put this site up. It went over the air with HTTP and no HTTPS! At the time, I had written only one webpage with basic html and my own css. I was incredibly happy to get my internet wire open to the world for the first time.

After coming back, I started 2019 by securing the line with HTTPS, and learning Bootstrap at a Milwaukee Hackathon. The first page I put together wtih bootstrap looked like this.
